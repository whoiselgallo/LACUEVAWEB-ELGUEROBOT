Escritorio — 1200px
Tablet — 768px
Móvil — 375px

Si los iframes no cargan por restricciones de "file://" en tu navegador, abre un servidor local desde la carpeta del proyecto:

# Python 3
python -m http.server 8000

# Luego abre en el navegador:
http://localhost:8000/preview-responsive.html

# En Windows (cmd):
start preview-responsive.html
